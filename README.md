# backloghq.io

Website and documentation for the [backlog](https://github.com/backloghq/backlog) Claude Code plugin.

Built with [Astro Starlight](https://starlight.astro.build). Deployed to GitHub Pages at [backloghq.io](https://backloghq.io).

## Development

```bash
npm install
npm run dev       # localhost:4321
npm run build     # output to dist/
```

## Deployment

Pushes to `main` auto-deploy via GitHub Actions.
