Captures your current progress so the next session can pick up seamlessly.

## What it does

1. Stops any actively tracked tasks
2. Annotates in-progress tasks with current status and context
3. Summarizes what was done, what's in progress, and what's next
4. Writes a handoff summary visible to the next session

## Usage

```
/backlog:handoff
```

Run this at the end of a work session. The annotations and summary persist in the task store, so the next session (or agent) has full context on where things stand.
