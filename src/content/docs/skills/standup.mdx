Generates a standup report from your task data.

## What it does

1. Shows tasks completed recently
2. Lists tasks currently in progress (active)
3. Identifies blocked tasks and what's blocking them
4. Suggests what to work on next based on priority and readiness

## Usage

```
/backlog:standup
```

Useful at the start of a session to orient yourself, or before a team sync to summarize progress.
