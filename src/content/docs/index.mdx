## Design principles

<CardGrid>
	<Card title="Agent-first">
		MCP servers, declarative schemas, and tools designed for how agents actually work — not retrofitted from human-facing libraries.
	</Card>
	<Card title="Zero native dependencies">
		Pure TypeScript. Runs anywhere Node.js does. No native binaries, no external databases.
	</Card>
	<Card title="Crash-safe by default">
		Append-only WAL, immutable snapshots, undo on every mutation. Tradeoffs are stated out loud, not hidden.
	</Card>
	<Card title="MIT, in the open">
		600+ tests per package, security and architecture reviews on each release. Built in public on GitHub.
	</Card>
</CardGrid>
