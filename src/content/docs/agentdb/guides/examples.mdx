## Quick Start

```bash
# Clone and build
git clone https://github.com/backloghq/agentdb.git
cd agentdb && npm install && npm run build

# Run a demo
cd examples/multi-agent && ./run.sh
```

## Key Patterns Demonstrated

| Pattern | Demo |
|---------|------|
| Event-driven coordination | Multi-agent, Research pipeline |
| Optimistic locking | Multi-agent (task claiming) |
| Hybrid search (BM25 + semantic) / RAG | RAG Knowledge Base |
| Multi-provider orchestration | Code Review (Gemini + Ollama) |
| Real-time monitoring | Live Dashboard |
| Pipeline via notifications | Research Pipeline, Code Review |
