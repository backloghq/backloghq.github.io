**Last updated: April 6, 2026**

## What backlog collects

Nothing. The backlog plugin runs entirely on your machine. No data is sent to any server.

## Data storage

All task data is stored locally in your filesystem:
- **Plugin mode**: `~/.claude/plugins/data/backlog/projects/<project-slug>/`
- **Standalone mode**: wherever you set the `TASKDATA` environment variable

## Data sharing

The plugin does not:
- Send data to any remote server
- Track usage or analytics
- Collect personal information
- Access your code, files, or any data beyond the task store

## Third-party services

The plugin has zero runtime dependencies beyond Node.js. It does not connect to any third-party service.

## Changes

If this policy changes, we will update this page with the new date.
