---
title: Skills Overview
description: Backlog skills at a glance.
---

Skills are slash commands that orchestrate multi-step workflows using the underlying MCP tools.

| Skill | Command | Purpose |
|-------|---------|---------|
| Tasks | `/backlog:tasks` | Show the current backlog: pending, active, blocked, overdue |
| Plan | `/backlog:plan` | Break down a goal into tasks with dependencies and specs |
| Standup | `/backlog:standup` | Daily standup: done, in progress, blocked, up next |
| Refine | `/backlog:refine` | Groom the backlog: fix vague tasks, missing priorities, broken deps |
| Spec | `/backlog:spec` | Write a detailed spec document for a task |
| Implement | `/backlog:implement` | Pick up a task, read its spec, implement it, mark done |
| Handoff | `/backlog:handoff` | Session handoff: annotate progress, stop active tasks, summarize |
